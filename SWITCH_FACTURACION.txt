===========================================================================
  SWITCH DEL MÓDULO DE FACTURACIÓN
  "AISLADO" ↔ "CONECTADO CON EL RESTO DEL SISTEMA"
===========================================================================

¿QUÉ ES ESTO?
--------------
HopeCare tiene 5 bases de datos separadas (auth, clinica, citas,
facturacion, dashboard). El módulo de facturación necesita leer datos
de otras bases (como citas y consultas) para saber qué está pendiente
de cobrar.

- MODO AISLADO = el módulo funciona SOLO, leyendo todo de una sola BD
                  (commit 21191ae, rama Modulo/facturar)
- MODO CONECTADO = el módulo lee datos de las otras BD del sistema
                   mediante ATTACH de SQLite (commit HEAD, rama modulosDebug)

CÓMO USAR ESTA GUÍA:
--------------------
Cada cambio está explicado como:

  ARCHIVO: ruta/del/archivo.java
  LÍNEA: XX
  AISLADO:   (texto cuando funciona solo)
  CONECTADO: (texto cuando está integrado)
  QUÉ HACE:  (explicación simple)

Si quieres AISLAR tu módulo: cambia CONECTADO → AISLADO
Si quieres RECONECTARLO: cambia AISLADO → CONECTADO
