# Guía Manual Exhaustiva: Integración del Sistema HopeCare

Este documento es una guía paso a paso y completamente manual para transformar el código del módulo "Aislado" (Pacientes y Médicos) en el sistema "Integrado" completo. **No se utilizarán comandos de Git.** Debes seguir cada paso, copiando y reemplazando el código exactamente como se indica.

---

## PASO 1: Preparación del Servidor MySQL

El sistema integrado no usa una sola base de datos, sino cinco. Abre tu gestor de base de datos MySQL (como Workbench, DBeaver o la consola) y ejecuta el siguiente script para crear los espacios necesarios:

```sql
CREATE DATABASE IF NOT EXISTS hopecare_clinica;
CREATE DATABASE IF NOT EXISTS hopecare_auth;
CREATE DATABASE IF NOT EXISTS hopecare_citas;
CREATE DATABASE IF NOT EXISTS hopecare_facturacion;
CREATE DATABASE IF NOT EXISTS hopecare_dashboard;
```

## PASO 3: Actualizar la Clase de Conexión

Abre el archivo: `src/main/java/com/esperanza/hopecare/util/DatabaseConnection.java`

Debes reemplazar TODO el contenido del archivo con este nuevo código, el cual asigna cada conexión a su base de datos específica en lugar de agrupar todo en una sola:

```java
package com.esperanza.hopecare.util;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DatabaseConnection {
    private static final String BASE_URL = "jdbc:mysql://localhost:3306/";
    private static final String USER = "root";
    private static final String PASSWORD = ""; // Asegúrate de que esta contraseña sea la tuya

    public static Connection getConnection(String dbName) throws SQLException {
        return DriverManager.getConnection(BASE_URL + dbName + "?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC", USER, PASSWORD);
    }

    public static Connection getClinicaConnection() throws SQLException {
        return getConnection("hopecare_clinica");
    }

    public static Connection getAuthConnection() throws SQLException {
        return getConnection("hopecare_auth");
    }

    public static Connection getCitasConnection() throws SQLException {
        return getConnection("hopecare_citas");
    }

    public static Connection getFacturacionConnection() throws SQLException {
        return getConnection("hopecare_facturacion");
    }

    public static Connection getDashboardConnection() throws SQLException {
        return getConnection("hopecare_dashboard");
    }

    @Deprecated
    public static Connection getConnection() throws SQLException {
        return getClinicaConnection();
    }
}
```

## PASO 5: Integración Lógica (La magia entre módulos)

Este es el cambio más importante. Cuando registramos un Médico en la Clínica, el sistema debe crearle automáticamente un horario en el módulo de Citas.

Abre el archivo: `src/main/java/com/esperanza/hopecare/dao/MedicoDAO.java`

Busca el método `public boolean insertarMedico(Medico m)`.  Justo antes de la línea `conn.commit();`, debes agregar el siguiente bloque de código. 

Tu método debería verse parecido a esto al final del bloque try principal:

```java
            // ... (código existente que inserta la persona y el médico) ...
            
            try (ResultSet rs = psM.getGeneratedKeys()) {
                if (rs.next()) {
                    m.setIdMedico(rs.getInt(1));
                }
            }
            
            // --- INICIO DEL NUEVO BLOQUE DE INTEGRACIÓN ---
            // Integración con el módulo de CITAS: Generar horario por defecto (Lunes a Domingo, 08:00 a 12:00)
            if (m.getIdMedico() > 0) {
                try (Connection connCitas = DatabaseConnection.getCitasConnection()) {
                    String sqlHorario = "INSERT INTO horario_atencion (id_medico, dia_semana, hora_inicio, hora_fin, intervalo_minutos, activo) VALUES (?, ?, ?, ?, ?, ?)";
                    try (java.sql.PreparedStatement psHorario = connCitas.prepareStatement(sqlHorario)) {
                        for (int dia = 1; dia <= 7; dia++) {
                            psHorario.setInt(1, m.getIdMedico());
                            psHorario.setInt(2, dia);
                            psHorario.setString(3, "08:00:00");
                            psHorario.setString(4, "12:00:00");
                            psHorario.setInt(5, 30); // 30 minutos de intervalo
                            psHorario.setInt(6, 1);
                            psHorario.addBatch();
                        }
                        psHorario.executeBatch();
                    }
                } catch (Exception ex) {
                    LOGGER.log(java.util.logging.Level.SEVERE, "Error al generar horario por defecto para el médico", ex);
                    // Decisión de diseño: ¿Frenamos la creación del médico si falla el horario? 
                    // En este caso, continuamos y dejamos el error en el log.
                }
            }
            // --- FIN DEL NUEVO BLOQUE DE INTEGRACIÓN ---

            conn.commit();
            return true;
```

## PASO 6: Limpieza Final y Arranque

1. Elimina cualquier dato residual en tu base de datos MySQL (opcional pero recomendado) ejecutando:
   `DROP DATABASE hopecare_clinica; DROP DATABASE hopecare_auth; DROP DATABASE hopecare_citas;`
2. Ejecuta el programa. Al iniciar, `HopecareApp` detectará que faltan las tablas, leerá los nuevos archivos separados (`clinica_schema.sql`, `auth_schema.sql`, etc.) y las creará en las 5 bases de datos diferentes.
3. Intenta crear un médico nuevo desde la interfaz. Si todo salió bien, el médico se guardará en `hopecare_clinica`, y su horario base aparecerá automáticamente en la tabla `horario_atencion` de `hopecare_citas`.
